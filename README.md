## 一个问题：和普通模拟器有什么区别？

很多人会拿安卓模拟器来对比，但本质上是两回事。

模拟器是在 x86 架构的 PC 上"模拟"ARM 手机，平台检测到指纹特征异常的概率更高，而且性能也受本地设备拖累。

DuoPlus 的底层是**真实 ARM 芯片设备部署在海外数据中心**，通过云端传输画面给你操作，设备行为和真手机完全一致——对平台来说，它就是一台真手机。

这是安全性上的根本差异，不是配置高低的问题。
